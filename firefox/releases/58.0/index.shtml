Firefox 新鮮事

  • 全新功能 效能改善：
    • Windows: 使用 Off-Main-Thread Painting (OMTP) 加速圖像描繪。
    • 調整 JavaScript 快取機制以提升網頁載入速度。
  • 全新功能 Firefox Screenshots 相關改進：
    • 直接複製、貼上到系統剪貼簿。
    • 在隱私瀏覽模式也能使用。
  • 全新功能 支援信用卡自動輸入功能（僅 en-US 語系）。
  • 全新功能 新增尼泊爾語（Nepali, ne-NP）語系支援。
  • 全新功能 為免您錯過—— 57 釋出的隱私與效能改善：使用者可以永遠啟用追蹤保護，請參考如何開啟追蹤保護。
  • 修正 Linux 上安裝於非預設目錄的字型顯示為空白的問題。
  • 修正 各種安全性修正。
  • 變更 以 Firefox 58 及未來版本建立的使用者設定檔將無法被舊版 Firefox 開啟。需要降級的使用者，請另外建立新的設定檔。參考降級與其他替代方案。
  • 變更 顯示關於 Symantec 憑證將被撤銷的警告，提醒受影響的使用者和管理者。
  • 開發功能 實作 PerformanceNavigationTiming API。
  • 開發功能 新增 PerformanceResourceTiming.workerStart，網站將可以測量 service worker 的啟動時間。
  • 開發功能 更多開發者功能改進。

已知問題

  • 尚未修復 使用 Windows 遠端桌面協定（RDP）執行的 Firefox 可能因為縮限的安全限制而無法播放聲音。在下個修正之前，可參考此問題的暫時性解法。
  • 尚未修復 使用螢幕閱讀器時可能有效能問題，建議在新版修正前，先暫時使用 Firefox ESR 版本。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
