Firefox 新鮮事

  • 全新功能 儲存密碼前，能檢視要儲存的密碼。
  • 全新功能 網址列新增縮放按鈕：
    • 縮放等級不是預設值（100%）時，顯示縮放等級。
    • 按下此按鈕即可恢復預設大小。
  • 全新功能 改善沒有 GPU 加速時的影像效能，減少 CPU 用量，增進全螢幕體驗。
  • 全新功能 即使表單沒有「submit」事件，也能使用記住密碼功能。
  • 全新功能 支援 FLAC (Free Lossless Audio Codec) 播放。
  • 全新功能 支援 WebGL 2，具備 transform feedback 等先進繪圖功能、更佳的材質處理能力，同時新增了精巧的 shading 語言。
  • 全新功能 若登入頁未使用安全連線，顯示警告訊息。
  • 全新功能 新增 Georgian (ka) 和 Kabyle (kab) 語系支援。
  • 全新功能 更快速的 E10s！改善分頁切換表現。
  • 全新功能 提升同步功能可靠度。
  • 全新功能 移除 Belarusian (be) 語系支援。
  • 變更 更新至 NSS 3.28.1。
  • 變更 (Linux) 採用 2D 繪圖引擎 Skia 進行內容繪圖。
  • 變更 重新啟用 Russian (ru) 語系的 E10s 支援。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
