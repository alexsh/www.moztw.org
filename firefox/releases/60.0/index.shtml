Firefox 新鮮事

  • 全新功能 以 Windows 群組原則或跨平台 JSON 設定檔，自訂部署 Firefox 企業版時的政策設定。
  • 全新功能 新分頁／Firefox 首頁相關改進：
    • 適應性設計，在寬螢幕顯示更多內容。
    • 精選網站將包括已儲存到 Pocket 的網頁。
    • 更多段落、內容用的排序選項。
    • 部分 US 使用者將開始看到 Pocket 贊助內容，參閱考量使用者隱私的贊助內容進行方式。
  • 全新功能 改進「Cookie 與網站資料」選項設計，更清楚呈現與控制第一方、第三方 cookie。
  • 全新功能 應用 Quantum CSS 描繪瀏覽器使用介面。
  • 全新功能 支援 Web Authentication API，允許採用 USB token 進行網站認證。
  • 全新功能 改善攝影機隱私權提示：禁用錄影功能時，Firefox 會將攝影鏡頭和燈光關閉，並於繼續錄影時再度開啟。
  • 全新功能 Linux: 新增是否顯示瀏覽器頂端標題列的選項。在瀏覽器主選單「自訂」面板中的「標題列」選項。
  • 全新功能 Linux: 改善 WebRTC 聲音效能。
  • 全新功能 預設即啟用最新 draft 版 TLS 1.3 支援。
  • 全新功能 新增奧克語（Occitan, oc）語系支援。
  • 全新功能 正體中文版: 新增 Readmoo 電子書為預設搜索引擎。
  • 修正 各種安全性修正。
  • 變更 進入閱讀模式的快速鍵改為 F9，與使用 AltGr 的鍵盤配置更為一致。
  • 變更 不再支援於同一 URL 的書籤設定複數關鍵字，除非該請求有不同的 POST 資料。
  • 變更 不再信任 2016 六月一日前由 Symantec 發行的 TLS 憑證。
  • 變更 升級 Skia 圖像引擎至 milestone 66。
  • 開發功能 參考開發者功能改進。

已知問題

  • 尚未修復 在新分頁停用 Sponsored Stories 後，下一次開啟分頁可能仍會顯示贊助磚塊（bug 1458906）。
  • 尚未修復 macOS 上使用 Vive 耳機組可能無法正常使用 WebVR（bug 1454204）。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
