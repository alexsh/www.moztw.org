Firefox 新鮮事

  • 全新功能 改善對有害下載檔案的防護。
  • 全新功能 開始對部分使用者啟用多程序模式（Electrolysis，e10s）。
  • 全新功能 未經 Mozilla 驗證簽署的附加元件將不會再被載入。
  • 全新功能 GNU/Linux: 支援 Skia 繪圖引擎以提升 canvas 效能。
  • 全新功能 WebRTC 改善：
    • 啟用 Delay-agnostic AEC。
    • 支援 GNU/Linux full duplex。
    • 支援 ICE Restart 與 Update。
    • 支援複製 MediaStream 和 MediaStreamTrack。
  • 全新功能 在網址列搜尋書籤或已開啟分頁時，會顯示相應的圖示。
  • 全新功能 Windows: Tab（移動焦點）和 Shift + F10（右鍵選單）在 Firefox 自訂模式中也能正常使用。
  • 全新功能 以 Rust 程式語言重新開發了媒體解析器。
  • 全新功能 Windows: 不能使用平台更新的系統，可以使用 D3D11 WARP。
  • 修正 各種安全性修正。
  • 修正 修正 Heyo、Jabra 和 Logitech C920 webcam 由於 WebRTC 臭蟲引起的雜音。
  • 修正 改善除錯工具 step 在函數最後一行時的行為。
  • 變更 Mac OS X: 停止支援 OS X 10.6、10.7 和 10.8，專注於 10.9 的支援；建議舊版使用者升級系統。
  • 變更 Windows: 此版本開始要求 CPU 支援 SSE2 延伸指令集。
  • 變更 停止支援 Windows Remote Access Service modem Autodial。
  • 開發功能 WebExtension 已進入穩定狀態。
  • 開發功能 Worker 可以使用 Web Crypto API。
  • 開發功能 在畫面上直接調整 position 定位元素的位移。
  • 開發功能 記憶體面板新增 Tree Map 檢視。
  • 開發功能 可以對 WebExtension 於背景執行的網頁或 content script 進行除錯。
  • 開發功能 WebExtension 開始強制執行 CSP (Content Security Policy)。
  • 開發功能 瀏覽器主控台取代了舊的「錯誤主控台」。
  • 開發功能 附加元件可以在 about:debugging 中重新載入，方便開發與除錯。
  • 開發功能 開發者工具新增「Firebug」佈景主題。
  • 開發功能 展開主控台中的網路請求，檢視詳細檔頭、回應、cookie 資訊。

已知問題
