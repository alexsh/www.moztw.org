Firefox 新鮮事

  • 參考 Firefox 新鮮事。
  • 修正 啟用硬體加速時，頁面描繪錯誤的問題（bug 1435472）。
  • 修正 Windows 7 輕鬆存取和某些應用程式一起使用時，可能造成瀏覽器當機，已知受影響的應用程式：StickyPassword、Windows 7 觸控螢幕（bug 1424505）。
  • 修正 使用 resistFingerprinting 的網站可能和瀏覽器快速鍵（例如 Ctrl + C 複製）衝突（bug 1433592）。
  • 修正 特定第三方軟體在某些電腦會大量耗用 CPU 與記憶體（bug 1446280）。
  • 修正 「選項 → 網路代理伺服器」畫面中，選擇「Proxy 自動設定網址」時無法點選「重新載入」（bug 1445991）。
  • 修正 URL fragment identifier 造成 Service Worker 回應錯誤（bug 1443850）。
  • 修正 在列印完成時嘗試取消列印，會斷斷續續地發生當機（bug 1441598）。
  • 修正 getUserMedia 在 DragonFly、FreeBSD、NetBSD、OpenBSD 無法取得聲音。聊天軟體無法運作，或只能靜音（bug 1444074）。
  • 修正 各種安全性修正。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
