Firefox 新鮮事

  • 全新功能 嶄新的瀏覽器引擎，致力於徹底運用現今電腦的計算能力。
  • 全新功能 重新設計使用者介面，具備簡潔時髦的外觀、一致性的視覺元素，同時對觸控螢幕進行最佳化。
  • 全新功能 整合網址列和搜尋列。新使用者將看見合併的網址列，您也能自行調整為分離的搜尋列。
  • 全新功能 更新「新分頁」頁面，包含最常造訪的、最近造訪的網頁和來自 Pocket（限美國、加拿大和德國版）的建議網頁。
  • 全新功能 更新給新使用者的導覽教學功能。
  • 全新功能 支援 AMD VP9 硬體加速，播放影片將消費更少電力。
  • 全新功能 選項畫面新增網站「權限」管理區塊。
  • 修正 各種安全性修正。
  • 變更 Firefox 現在只支援採用 WebExtenstion API 的附加元件，不再支援舊式元件。請參考我們為提升元件效能和安全性所做的努力。
  • 變更 自動捲動、鍵盤操作的捲動、觸控螢幕上的拖曳捲動現在都會以非同步方式處理，提供類似滑鼠滾輪等操作般的順暢體驗。
  • 變更 Linux 上的內容處理程序採用了更嚴格的安全沙箱，將阻擋對檔案系統的讀寫，如同 Firefox 56 在 Windows 和 macOS 提供的保護。
  • 變更 Unix 系統預設行為調整：在網頁內容使用滑鼠中鍵貼上時，不會直接前往該網址。
  • 變更 移除工具列中的「分享」按鈕。仍需要的使用者可自行安裝 Share Backported 擴充套件。
  • 變更 在 64 位元 Windows 停用有當機問題的舊版 ATOK IME（包括 ATOK 2006、2008、2009 和 2010），請安裝新版 ATOK 或其他 IME 以解決此相容問題。
  • 變更 預設日文字型改為 Meiryo。
  • 開發功能 伴隨 Firefox Quantum 視覺設計，翻新開發者工具亮色／暗色佈景主題。
  • 開發功能 檢測器中的 CSS 變數，在滑鼠移上時會顯示變數的值。
  • 開發功能 重新設計主控台面板，和除錯器、網路監測器一樣採用 React、Redux 技術重寫，同時新增檢測物件各層內容的功能。
  • 開發功能 更多開發者功能改進。

已知問題

  • 尚未修復 使用 Windows 遠端桌面協定（RDP）執行的 Firefox 可能因為縮限的安全限制而無法播放聲音。在下個修正之前，可參考此問題的暫時性解法。
  • 尚未修復 使用螢幕閱讀器時可能有效能問題，建議在新版修正前，先暫時使用 Firefox ESR 版本。
  • 尚未修復 在 Windows 和 Linux 使用 Intel Broadwell-U 處理器配合舊版 microcode 時有機會當機。Windows 用戶請開啟 Windows Update，Linux 用戶請安裝目前發佈版本提供的 Intel microcode。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
