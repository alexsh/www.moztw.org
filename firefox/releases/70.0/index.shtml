Firefox 新鮮事

  • 全新功能 加強型追蹤保護（ETP） 推出更多隱私保護功能：
    • 社交追蹤保護功能現在已納入加強型追蹤保護的標準功能，它能封鎖跨站追蹤的 Cookie，像是來自 Facebook、Twitter 和 LinkedIn。
    • 隱私保護報告可概觀顯示 Firefox 封鎖的追蹤器，也可提供詳細資訊；其提供的綜合報告來自 Firefox Monitor 和 Lockwise。
  • 全新功能 我們的數位身分和密碼管理工具 Firefox Lockwise 提供更多安全保護：桌面版的 Lockwise 讓您的密碼與登入可同步到所有裝置囉！舉凡 Lockwise 手機 APP 和 Firefox 手機版瀏覽器，都可輕鬆創建、更新或刪除登入記錄和密碼。
  • 全新功能 Firefox Monitor 的綜合外洩警報 (Integrated breach alerts) 在儲存的登入資訊和密碼等線上數據外洩時會發出警示提醒。
  • 全新功能 在網站上註冊新帳號時，可透過複雜密碼產生器 (Complex password generation) 來自動產生並儲存強密碼。
  • 全新功能 為提升網站瀏覽體驗，核心引擎零件改善了以下功能：
    • Javascript Baseline 直譯器處理現代化網頁的龐大程式庫更快了！網頁載入速度加快率高達 8%。
    • 為更多 Windows 的 Firefox 使用者開啟 WebRender 功能：現在起，使用 Intel 內建顯示卡，螢幕解析度又小於 1920x1200 的 Windows 桌上型電腦使用者，將獲得更好的算繪效果。
    • macOS 版的 Firefox 排版改善，上網更省電，網頁載入速度加快 37%。
  • 全新功能 Firefox 新產品與服務帶來更多的瀏覽器功能：
    • 獨立運作的 Firefox 帳戶選單能讓 Firefox Monitor 和 Firefox Send 等服務操作更簡便。
    • 工具列上的禮物圖示訊息提示能迅速呈現新版本當中有哪些新鮮事。
    • 如果網站需要存取您的所在位置，會在網址列跳出提示告知您。
  • 修正 各種安全性修正。
  • 變更 Firefox 的內建頁面現在可以依您系統設定的偏好，以夜間模式呈現囉。
  • 變更 佈景主題的別名屬性將被移除，有些佈景主題可能因此受到影響。
  • 變更 現在從 macOS 上的 Chrome 瀏覽器和 Windows 都可以匯入密碼囉。
  • 變更 文字上方或下方畫有上方線或底線者，像是超連結文字，其可讀性已獲得顯著改善了！這些線會自動略過字符本身，不會直接畫在字上。
  • 變更 隱私與安全性標示再進化
    • 不安全的 HTTP 網站上會顯示全新的「斜槓鎖頭」圖示
    • 以前全綠色的鎖頭圖示現在改成灰色的囉
    • 擴充驗證（The Extended Validation）指標現在移到身份驗證框上了，當您點擊鎖頭圖示，身份驗證框就會彈出來囉
  • 開發功能 如需閱覽 Mozilla 的附加開發功能資源，歡迎每週收看我們的 Mozilla 開發者 Youtube 頻道最新的影片。
  • 開發功能 開發者工具的「輔助功能」面板已可檢查鍵盤操作的易用性，也將色弱模擬器納入使用 Firefox WebRender 的系統中。
  • 開發功能 未啟用的 CSS：不對任何元素生效的 CSS 顯示為灰色之外，檢測器會一併提示原因及修復方法。
  • 開發功能 開發者工具中的新 DOM Mutation 中斷點能在指令碼新增、移除或更新頁面內容時協助判斷問題，能讓複雜的指令碼互動和除錯輕鬆許多！。
  • 開發功能 WebExtensions 開發者現在可以用 Firefox 開發者工具的附加元件除錯工具箱，檢查 browser.storage.local 的資料囉。
  • 開發功能 使用開發者工具中的網路監視器，您就能迅速找到資源的請求與回應資料，包含標頭、cookie 和內容。
