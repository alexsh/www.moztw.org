Firefox 新鮮事

  • 全新功能 Windows: 分離影像合成程序（採用 Quantum Compositor）以改善效能，並減少圖像顯示相關的當機情形。
  • 全新功能 新增「精簡」亮色、暗色佈景主題（基於 Firefox 開發者版本的設計）。
  • 全新功能 佈景主題（lightweight theme）現在也能套用在隱私視窗。
  • 全新功能 閱讀模式會顯示預估的閱讀時間。
  • 全新功能 Windows 7 以上的 64 位元系統中，安裝程式會提供 32 或 64 位元的版本選擇。
  • 修正 各種安全性修正。
  • 變更 改善網站要求權限的通知介面，減少被輕易忽略的狀況。
  • 變更 不再支援 Windows XP 和 Vista。但會持續提供延長支援版本（ESR）的安全性更新給 XP 和 Vista 的 Firefox 52 使用者。
  • 變更 不再支援 32 位元的 Mac OS X。舊使用者可以轉移至 Firefox ESR 52。
  • 變更 減少 Mac OS X 更新程式的檔案大小。
  • 變更 影片、聲音控制介面有了新的視覺設計。
  • 變更 Linux: 停止支援低於 Pentium 4 和 AMD Opteron 的處理器。
  • 變更 分頁標題過長的部分，使用淡出效果而非省略符號，保留較多可見文字。
  • 開發功能 更多開發者功能改進。
  • 開發功能 支援帶有 alpha 頻道的 WebM 影像，可以播放具有透明背景的影片。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
