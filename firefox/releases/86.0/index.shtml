Firefox 新鮮事

  • 在 Mozilla，我們相信您有隱私權。您不該在上線時被追蹤，不論您正在查看銀行餘額、尋找最好的醫師或者買鞋子，不道德的追蹤公司不該有辦法在您瀏覽網路時追蹤您。因此，我們持續強化 Firefox，以替我們的使用者抵抗線上追蹤。
  • 全新功能 Firefox 現在 支援 以 Picture-in-Picture 模式同時觀賞 數個影片。
  • 全新功能 今天，Firefox 加入了 Total Cookie Protection 到 Strict 模式中。 在 Total Cookie Protection 裡，每個網站都會有它自己的 "cookie jar"， 避免您在瀏覽不同網站時被追蹤.
  • 全新功能 我們現在改善了網頁列印功能，有更乾淨的設計並更好的整合了您電腦上的印表機設定。（這邊有一張圖要加）
  • 全新功能 現在加拿大的 Firefox 使用者來可以使用 信用卡管理及自動填入功能 了
  • 修正 Reader mode now works with local HTML pages.
  • 修正 Using screen reader quick navigation to move to editable text controls no longer incorrectly reaches non-editable cells in some grids such as on messenger.com.
  • 修正 The Orca screen reader's mouse review feature now works correctly after switching tabs in Firefox.
  • 修正 Screen readers no longer report column headers incorrectly in tables containing cells spanning multiple columns.
  • 修正 Links in Reader View now have more color contrast.
  • 修正 修補幾種不同的安全弱點。
  • 變更 On Linux and Android, the protection to mitigate the stack clash attack has been activated.
  • 變更 From Firefox 86 onward, DTLS 1.0 is no longer supported for establishing WebRTC's PeerConnections. All WebRTC services need to support DTLS 1.2 from now on as the minimum version.
  • 變更 Consolidated all video decoding in the new RDD process which results in a more secure Firefox.
  • 企業版 多個 Bug 修復與新政策加入到最新的 Firefox 版本。您可在 Firefox 86 企業版版本資訊找到更多資訊。
  • 開發功能 CSS image-set() function in CSS is now enabled, allowing for responsive images in CSS.
  • 開發功能 Inactive CSS tool is now showing a warning when margin or padding is set on internal table elements. 這邊有一張圖要加
  • 開發功能 Developer Tools Toolbox is now showing a number of errors on the current page. This is a quick way to surface information to a developer that something is wrong with their page. Clicking on the red exclamation icon navigates the user to the Console panel. 這邊有一張圖要加
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
