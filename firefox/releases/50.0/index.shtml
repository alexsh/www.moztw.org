Firefox 新鮮事

  • 全新功能 關於鍵盤快速鍵的更新：
    • 可設定按下 Ctrl+Tab 時，依照最近使用分頁的順序切換分頁標籤。
    • 按下 Ctrl+Alt+R（在 Mac 是 command+alt+r）以閱讀模式檢視網頁。
  • 全新功能 頁面中搜尋功能可以只尋找符合「整個文字」的項目。
  • 全新功能 新增 Guarani (gn) 語系支援。
  • 全新功能 提升 Windows 7 以上系統的 WebGL 支援，讓 WebGL 普及至 98% 的使用者。
  • 全新功能 下載檔案防護功能支援更多 Windows、Mac 和 Linux 的可執行檔類型。
  • 全新功能 改善 SDK 擴充元件（或使用 SDK module loader 的擴充元件）效能。
  • 全新功能 Windows 和 Mac 可藉由 Widevine 的 WebM EME 支援在更多網站上播放影片，而不需要外掛程式。
  • 修正 修正虛線、點線在圓角部位的描繪（見 border-radius）。
  • 修正 各種安全性修正。
  • 修正 在使用大量 cookie 的網站也能正確儲存登入 cookie （bug 1264192）。
  • 變更 新增一組內建 Emoji，供沒有原生 Emoji 字型的系統（Windows 8.0 以下和 Linux）使用。
  • 變更 停用 54.35.1 之前的舊版 libavcodec。
  • 開發功能 見 Changes for web developers。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
