Firefox 新鮮事

  • 全新功能 效能改善：
    • 提升 Firefox 首頁的內容載入進度。
    • 由網路／硬碟之一載入快取（Race Cache With Network），加速頁面讀取。
    • Mac 系統也採用 Off-Main-Thread Painting (OMTP) 改善圖像描繪速度。（Windows 版已於 Firefox 58 釋出）。
  • 全新功能 拖曳排序 Firefox 首頁的熱門網站，及其他關於新視窗、分頁的客製化。
  • 全新功能 Firefox Screenshots 的新功能：
    • 使用筆和螢光筆工具在已儲存擷圖上標記。
    • 裁切已儲存的擷圖。
  • 全新功能 WebExtensions API 相關改進，包括對去中心化通訊協定的支援、動態註冊內容腳本等。
  • 全新功能 Real-Time Communications (RTC) 相關改進：
    • 實作 RTP Transceiver，可以對各個呼叫進行更細膩的控制。
    • 實作用於支援大規模會議的功能。
  • 全新功能 支援 W3C 標準的 pointer events，改善各平台滑鼠、觸控筆和觸控螢幕等指標裝置輸入的整合。
  • 全新功能 德語版新增 Ecosia 搜尋引擎。
  • 全新功能 法語版新增 Qwant 搜尋引擎。
  • 全新功能 增加可以阻擋特定網站要求通知、攝影機、麥克風、位址等權限的 about:preferences 選項。
  • 修正 各種安全性修正。
  • 變更 隱私瀏覽模式將移除 referrer 中的路徑資訊，以避免跨網站追蹤。

已知問題

  • 尚未修復 Windows 7 輕鬆存取功能（例如螢幕小鍵盤）可能在升級 Firefox 59 後引起瀏覽器當機，受影響的使用者可參考此問題的暫時性解法（阻止外部程式在 Firefox 中取用輕鬆存取功能）。
