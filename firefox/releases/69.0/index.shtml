Firefox 新鮮事

  • 全新功能 加強型追蹤保護：
    • 現在起，此功能的標準設定（預設）會封鎖第三方追蹤 cookie 和加密貨幣採礦程式。
    • 而選用的嚴格設定，除了封鎖標準設定的項目外，還會封鎖數位指紋追蹤程式。
  • 全新功能 封鎖自動播放：除了自動播放聲音的影片以外，現在還能選擇封鎖任何影片自動播放。
  • 全新功能 即將推出嶄新的「新分頁」內容，呈現最棒的 Pocket 內容（僅針對美國或使用美式英語版本推出）。
  • 全新功能 Windows: 支援 Windows Hello 的 Web Authentication Hmac Secret 擴充功能，使用 2019 年 5 月 10 日後釋出的 Windows 即可支援無密碼上網體驗。
  • 全新功能 這一版將支援多種視訊解碼器，網路即時通訊會議服務整合不同客戶的影像時將更趨便利。
  • 全新功能 針對 Windows 10，推出更多效能與介面改善：
    • Firefox 會提供資訊給 Windows 以針對內容程序更準確地設定優先順序。也就是說，處理器會花更多時間在您正在主動操作的部分，而花更少時間在背景分頁中（影音播放除外）。
    • 還可以在 Win10 的工作列上輕鬆找到 Firefox 的捷徑。
  • 全新功能 針對 macOS 使用者，改善了電力消耗與檔案下載介面：
    • 於雙顯示卡裝置（例如 MacBook Pro）將更積極地使用較低功率的顯示卡，有助提升續航力與電池壽命。
    • Finder 現在也能在檔案下載的同時顯示進度了。
  • 全新功能 為改善 JavaScript 最佳化 JIT 編譯器性能，現在 JIT 已經可以支援 ARM64 了。
  • 修正 各種安全性修正。
  • 變更 根據 Firefox 先前宣布的 Plugin Roadmap，已移除 Flash 外掛程式的「總是啟用」選項。現在起每次都會先徵求使用者許可，才播放網站上的 Flash 內容。
  • 變更 不再採用 Adobe Flash 播放器以來，Firefox 也不再需要驗證使用者是否在 64-bit 作業系統上跑 32-bit 的 Firefox，減少使用者代理和數位指紋等並提供更進階的隱私同時，下載其他應用程式的體驗也獲得了改善。
  • 變更 Firefox 不再提供預設載入 userChrome.css 和 userContent.css 以增進啟動性能，想使用前述文件來自訂 Firefox 的使用者可以將 toolkit.legacyUserProfileCustomizations.stylesheets preference 設定為 true，就能重啟此種功能。
  • 企業版 管理 macOS 電腦的企業版系統管理員們，我們開始推出經 Mozilla 簽署的 package 安裝包，可簡化佈署過程。
  • 開發功能 我們的行動網頁開發者們，我們已將 WebIDE 遠端除錯功能搬到全新設計的about:debugging，可藉由 USB 穩當當地在遠端對 GeckoView 除錯。
  • 開發功能 網路資訊面板現在可以顯示封鎖的資源，讓開發者充分感受到內容封鎖功能的威力，還有這版能讓我們為所有使用者送上持續進化的加強型追蹤保護功能。
  • 開發功能 全新的事件監聽器的中斷點功能可讓開發者暫停在不同事件類型的 host，舉凡動畫、DOM、媒體、滑鼠、觸控、Worker，以及其他許多事件類型。
  • 開發功能 Firefox 開發者工具現在起可針對非文字內容檢查是否包含替代文字內容，輔助功能面板可檢查工具列是否已加入輔助內容，以協助開發者更完整遵守 WCAG Guideline 1.1。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
