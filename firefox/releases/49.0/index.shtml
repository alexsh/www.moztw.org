Firefox 新鮮事

  • 全新功能 允許 HTTPS 網站使用已儲存的 HTTP 登入資訊。這是 Firefox 支持 Let’s Encrypt 計畫、迎向更安全網路世界的方式之一。
  • 全新功能 「閱讀模式」視與聽的改善：
    • 字體控制：可以調整文字寬度和行高。
    • 朗讀：大聲唸出網頁內容。
  • 全新功能 沒有硬體加速時，支援以 SSSE3 改善影片處理效能。
  • 全新功能 HTML5 audio 與 video 右鍵選單新增「循環」和 1.25 倍速播放功能。
  • 全新功能 Mac OS X：
    • 改善無硬體加速時的效能。
    • 改善反鋸齒字型的顯示。
  • 全新功能 about:memory 能報告字型相關的記憶體用量。
  • 全新功能 Windows: 改善無硬體加速時的效能。
  • 修正 Mac: 修正原始 Firefox 安裝者以外的使用者，無法更新 Firefox 的問題。修改為具有管理權限的使用者皆能更新 Firefox。
  • 修正 各種安全性修正。
  • 變更 終止對 OS X 10.6、10.7 和 10.8 的支援。
  • 變更 終止對 Windows SSE 處理器的支援。
  • 變更 移除 Firefox Hello。
  • 變更 重新啟用 Graphite2 字體成型元件庫預設值。
  • 開發功能 網路監控器新增「原因」欄位，顯示網路請求的由來。
  • 開發功能 導入 web speech synthesis API。
  • 請參考此版本的完整變更清單。 您可能也想瞭解前一個版本的更新。
