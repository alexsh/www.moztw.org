Thunderbird 新鮮事

  • 新增 資料夾窗格工具列與資料夾畫面選擇器（取代資料夾畫面中的箭頭）。
  • 新增 從 Thunderbird 移除帳號時，亦可刪除對應的資料檔案。
  • 新增 可從 Becky! Internet Mail 匯入設定。
  • 新增 可複製訊息過濾器。
  • 新增 編輯草稿時會恢復字典設定。
    訊息中會包含 Content-Language 檔頭（RFC 3282）。
  • 新增 行事曆：現在起可在分頁中建立或編輯事件。
  • 新增 行事曆：可針對收到的邀請處理反對提議。
  • 新增 聊天：支援 Twitter 的私訊功能（Direct Messages）。
  • 新增 聊天：對 Twitter 的推文打心或加入最愛。
  • 新增 聊天：XMPP：支援 SASL SCRAM 驗證機制。
  • 新增 聊天：支援 Jabber/XMPP 的訊息副本功能（XEP-280）。
  • 變更 重要：圖片放入編輯視窗的行為有所異動。現在起將以 Data URI 的方式加入圖片，而不參照到其他訊息當中的內容或作業系統中的檔案。這個行為可與 MS Office 或 LibreOffice 等辦公室軟體有更好的互操作性。
    將不再下載或自動附加網際網路上的圖片。可以於 mail.compose.attach_http_images 偏好設定統一調整，或是逐一到每張圖片的「圖片屬性」對話框調整此行為。
  • 變更 所有新信件匣將預設顯示收件者欄位，可使用 mail.threadpane.use_correspondents 偏好設定關閉。
  • 變更 在 Linux 上必須安裝 PulseAudio 才可播放音效。
  • 變更 現在起，郵件傳遞格式切換成純文字時，格式化工具列將保留於原地。
  • 變更 除非將 mailnews.allowMboxOver4GB 偏好設定設為 false，否則將直接支援大於 4GB 的 mbox 格式信件匣儲存空間，不顯示警告。
  • 變更 IMAP 快取將使用 Mozilla 最新的快取技術。
  • 變更 於編輯視窗中插入超鏈結的快速鍵從 CTRL+L 更改為 CTRL+K，以與其他辦公室套裝軟體相容。
  • 變更 由於 Yahoo 停止即時通服務，故移除相關功能。
  • 修正 IMAP 信件匣更名或移動後，訊息預覽窗格無法使用。
  • 修正 編輯段落格式時，有時按下 Shift+Enter 無法換行。
  • 修正 修正以段落格式撰寫訊息時的許多問題。
  • 修正 「以引言方式貼上」功能有時無法使用。
  • 修正 以純文字格式回覆較長的文字時不會正確換行。
  • 修正 段落格式簽名前出現多於的空白。
  • 修正 無法使用附件時，撰寫視窗出現無限的「附加中…」而非顯示錯誤。
  • 修正 有時回覆的文字編碼有時會不正確（使用檢視上一封訊息使用的編碼）。
  • 修正 顯示、回覆或轉寄訊息的文字編碼有時會不正確（使用附件的編碼）。
  • 修正 儲存草稿時不會保留傳送格式（自動偵測、純文字、HTML、兩者皆有）。
  • 修正 未使用正確身分回覆郵件給自己。
  • 修正 IMAP 訊息部分快取。
  • 修正 無法點擊使用跳脫的非 ASCII（國際）字元超鏈結。
  • 修正 行事曆：指定於「本地時間」時區的事件，會於 UTC 時間產生警示。
  • 修正 聊天：XMPP 資源衝突。
  • 修正 許多安全性修正。

已知問題

  • 尚未修復 某些情況下會印出大量空白頁面。要修正這個問題，請到設定編輯器中清除所有印表機相關偏好設定。
  • 尚未修復 若訊息儲存在沒有同步供離線使用的 IMAP 信件匣，大型附件可能不會正常顯示或儲存。暫時替代方法：將 browser.cache.memory.max_entry_size 偏好設定設為 -1（不限制），並建立整數型偏好設定 browser.cache.memory.capacity，將值設定為 200000（200 MB，每封訊息 25 MB）。
  • 尚未修復 與 McAfee Anti-SPAM 附加元件不相容，發生程式錯誤。暫時替代方法：以安全模式開啟，並停用 McAfee Anti-Spam 擴充套件。
  • 尚未修復 無法使用背景圖片，以及撰寫郵件時與嵌入圖片相關的其他問題。
  • 尚未修復 壓實重整 IMAP 信件匣時發生程式錯誤。
